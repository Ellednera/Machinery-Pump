NAME

Machinery::Pump - A general pump

VERSION

Version 0.01

SYNOPSIS

Quick summary of what the module does.

Perhaps a little code snippet.

use Machinery::Pump;

my $foo = Machinery::Pump->new();
...

SUBROUTINES/METHODS

new( \%options )

Creates a new pump. For %options, only pump_id is required, see below for more details.

designed_flowrate => $numerical

The designed flowrate of the pump

rps => $integer

The blade spin rate in round per second

remote_control => $bool

The ability to control the pump through the DCS

status = 0

The status of the pump. This value is always set to 0 upon creation to indicate it is not running. This value can be changed by calling the status method.

pump_id

Returns the pump id

designed_flowrate( [ $numerical ] )

Gets or sets the designed flowrate, the unit is assumed to be m^3 / h

rps( [ $integer ] )

Gets of sets the rps, the unit is assumed to be in seconds

remote_control( [ "yes" | "no" ] )

Gets of sets the ability to control the pump remotely.
